一、等效原理

爱因斯坦在 1915 年提出的广义相对论，将引力描述为时空的几何属性。等效原理指出：局部范围内，引力场与加速参考系是不可区分的。

Gμν + Λgμν = 8πG / c⁴ · Tμν

其中 Gμν 是爱因斯坦张量，gμν 是度规张量，Tμν 是能量-动量张量，Λ 为宇宙学常数。

二、测地线方程

  • 自由粒子沿测地线运动
  • 测地线方程由 Γ Christoffel 符号决定
  • 弱场近似下退化为牛顿引力
d²xμ/dτ² + Γμαβ (dxα/dτ)(dxβ/dτ) = 0

思考：当 Λ ≠ 0 时，宇宙正在加速膨胀，这是 1998 年天文观测的重大发现。

场方程的求解需要考虑对称性，球对称真空解即 Schwarzschild 解：（建议补充 Schwarzschild 度规表达式）⇥

ds² = −(1 − 2GM/rc²) dt² + (1 − 2GM/rc²)−1 dr² + r²dΩ²
