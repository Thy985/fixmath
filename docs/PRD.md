# FormulaFix 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品简介

| 项目 | 内容 |
|------|------|
| **产品名称** | FormulaFix |
| **产品类型** | 移动端文档处理工具 (Android) |
| **核心定位** | 纯本地运行的 Markdown/LaTeX 编辑器，支持公式渲染与多格式导出 |
| **目标用户** | 教育行业从业者（老师、学生） |
| **核心价值** | 离线、高效、保护隐私的数学文档处理工具 |

### 1.2 产品愿景

打造一款**无服务器依赖、100%离线运行**的移动端文档处理工具，让用户可以随时随地编辑包含数学公式的 Markdown 文档，并导出为 PDF/Word 格式。

---

## 2. 功能需求

### 2.1 MVP 阶段（核心功能）

#### 2.1.1 输入模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 文本输入 | 支持直接粘贴/输入 Markdown 源码 | P0 |
| 文件导入 | 支持从手机存储导入 .md 文件 | P1 |
| 剪贴板监听 | App 启动时检测剪贴板，提示是否导入 | P2 |
| 本地保存 | 将编辑内容保存为 .md 文件到本地 | P1 |

#### 2.1.2 渲染引擎（核心）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| Markdown 解析 | 支持标题、列表、引用、代码块、表格 | P0 |
| LaTeX 行内公式 | 支持 `$...$` 格式的行内公式渲染 | P0 |
| LaTeX 块级公式 | 支持 `$$...$$` 格式的块级公式渲染 | P0 |
| 实时预览 | 编辑内容实时渲染，支持滚动查看 | P0 |

#### 2.1.3 导出模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| PDF 导出 | 保持排版格式，公式清晰，支持矢量渲染 | P0 |
| Word 导出 | 导出 .docx 格式，公式转为高清图片 | P1 |
| 分享功能 | 调用系统分享菜单，分享导出文件 | P1 |

### 2.2 迭代阶段（进阶功能）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| Mermaid 图表 | 支持 Mermaid 代码渲染为图片 | P2 |
| 深色/浅色模式 | 支持深色/浅色主题切换 | P2 |
| 模板系统 | 提供试卷模板、论文模板 | P2 |
| 历史版本 | 支持文档历史记录与回滚 | P3 |

## 3. 非功能需求

### 3.1 性能要求

| 指标 | 要求 |
|------|------|
| 首屏加载时间 | < 2 秒 |
| 公式渲染延迟 | < 100ms（单个公式） |
| 文档预览响应 | < 500ms |
| PDF 导出时间 | < 5 秒（10页以内） |
| Word 导出时间 | < 10 秒（10页以内） |

### 3.2 兼容性要求

| 项目 | 要求 |
|------|------|
| Android 版本 | Android 6.0 (API 21) 及以上 |
| 屏幕适配 | 支持手机和平板 |
| 存储空间 | 预留 100MB 安装空间 |

### 3.3 数据安全

| 要求 | 说明 |
|------|------|
| 离线运行 | 100% 本地处理，无网络请求 |
| 数据隐私 | 用户数据不离开设备 |
| 文件安全 | 使用应用私有目录存储数据 |

## 4. 用户流程

### 4.1 核心用户流程

```
┌──────────────────────────────────────────────────────────────┐
│                        用户流程                              │
└──────────────────────────────────────────────────────────────┘

[启动 App]
    │
    ▼
[检查剪贴板] ──有内容──→ [弹窗提示导入] ──确认──→ [填充到编辑器]
    │                              │
    │                              └──取消──→ [空编辑器]
    │
    ▼
[空编辑器]
    │
    ▼
[用户输入 Markdown/LaTeX]
    │
    ▼
[实时预览渲染]
    │
    ├──[继续编辑]─────────────────────────────────────────────
    │
    ▼
[点击导出]
    │
    ├──[PDF]──→ [生成 PDF]──→ [系统分享]
    │
    └──[Word]──→ [生成 Word]──→ [系统分享]
```

### 4.2 关键页面

| 页面 | 描述 |
|------|------|
| 首页/编辑器 | 主编辑页面，包含输入区和预览区 |
| 导出设置 | 选择导出格式、文件名等 |
| 文件管理 | 查看和管理本地保存的文档 |
| 设置 | 主题切换、关于信息 |

## 5. 验收标准

### 5.1 MVP 验收标准

| 功能 | 验收条件 |
|------|----------|
| 文本输入 | 可以正常输入和编辑 Markdown 文本 |
| Markdown 渲染 | 标题、列表、引用、代码块、表格正确显示 |
| 行内公式 | `$E=mc^2$` 正确渲染为公式 |
| 块级公式 | `$$\int_0^1 x^2 dx$$` 正确渲染为居中公式 |
| PDF 导出 | 导出的 PDF 包含公式，排版正确 |
| Word 导出 | 导出的 Word 文档可打开，公式显示为图片 |
| 分享功能 | 可以调用系统分享菜单分享文件 |

### 5.2 迭代阶段验收标准

| 功能 | 验收条件 |
|------|----------|
| Mermaid 图表 | Mermaid 代码正确渲染为图片 |
| 深色模式 | 切换主题后界面正确显示 |
| 文件导入 | 可以从手机存储导入 .md 文件 |
