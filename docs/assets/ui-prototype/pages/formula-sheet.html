广义相对论场方程推导

爱因斯坦场方程是广义相对论的核心方程，描述了引力如何由时空曲率产生。在这篇笔记中，我们从等效原理出发，逐步推导出真空场方程的形式。

1915 年 11 月 25 日，爱因斯坦向普鲁士科学院提交了最终形式的引力场方程。该方程将时空的几何结构（左侧的爱因斯坦张量）与其中的物质能量分布（右侧的能量-动量张量）联系起来。

Schwarzschild Metric
ds² = −(1 − 2GM/rc²) c²dt² + (1 − 2GM/rc²)⁻¹ dr² + r²(dθ² + sin²θ dφ²)
$$ ds^2 = -\left(1-\frac{2GM}{rc^2}\right)c^2 dt^2 + \left(1-\frac{2GM}{rc^2}\right)^{-1} dr^2 + r^2 d\Omega^2 $$

史瓦西解是场方程在球对称、静态、真空条件下的第一个精确解，由卡尔·史瓦西于 1916 年在第一次世界大战的战壕中得到。它描述了一个不旋转、不带电的球对称天体外部的引力场。

当半径 r 趋近于史瓦西半径 r_s = 2GM/c² 时，度规的时间分量趋于零，空间分量发散，这标志着事件视界的形成。

Gμν + Λgμν = (8πG/c⁴) Tμν

上述方程中，Gμν 是爱因斯坦张量，gμν 是度规张量，Λ 是宇宙学常数，Tμν 是能量-动量张量。这个方程的简洁性掩盖了其背后深刻的物理内涵。
