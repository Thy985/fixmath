# FormulaFix UI/UX 设计文档

## 1. 设计原则

### 1.1 核心设计原则

| 原则 | 说明 |
|------|------|
| **简洁高效** | 减少操作步骤，快速完成任务 |
| **所见即所得** | 编辑内容实时预览，减少切换 |
| **移动优先** | 优化触控交互，适合单手操作 |
| **内容为王** | 突出文档内容，减少视觉干扰 |

### 1.2 设计风格

| 维度 | 选择 |
|------|------|
| **视觉风格** | 简洁现代，类似 Notion/语雀 |
| **配色方案** | 蓝紫色调，专业且舒适 |
| **字体** | 优先系统字体，确保兼容性 |
| **图标** | 线性图标，轻盈现代 |

## 4. 页面设计

### 4.1 首页（编辑器页面）

```
┌─────────────────────────────────────────┐
│  FormulaFix                        ⋮    │  ← 顶部导航栏
├─────────────────────────────────────────┤
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ ## 第一章 函数的极限             │   │
│  │                                 │   │
│  │ 定义：设函数 f(x) 在点 x₀ 的    │   │
│  │ 某一去心邻域内有定义，若存在    │   │  ← 编辑区域
│  │ 常数 A，对于任意给定的正数 ε，  │   │    (TextField)
│  │ 总存在正数 δ，使得当           │   │
│  │ 0 < |x - x₀| < δ 时，          │   │
│  │ 都有 |f(x) - A| < ε 成立，    │   │
│  │ 则称 A 为函数 f(x) 当 x 趋向   │   │
│  │ 于 x₀ 时的极限，记作           │   │
│  │                                 │   │
│  │ $$\lim_{x \to x_0} f(x) = A$$  │   │  ← 公式预览
│  │                                 │   │
│  └─────────────────────────────────┘   │
│                                         │
├─────────────────────────────────────────┤
│  [预览] [保存] [导出 ▼]                  │  ← 底部操作栏
└─────────────────────────────────────────┘
```

## 6. 交互设计

### 6.1 手势操作

| 场景 | 操作 | 效果 |
|------|------|------|
| 编辑区域 | 单指滑动 | 滚动内容 |
| 编辑区域 | 双指捏合 | 调整字体大小 |
| 预览区域 | 双击公式 | 放大查看 |
| 底部栏 | 上滑 | 展开导出菜单 |
| 列表项 | 左滑 | 删除（带确认）|

### 6.2 动画效果

| 场景 | 动画 | 时长 |
|------|------|------|
| 页面切换 | 滑动淡入 | 300ms |
| 按钮点击 | 缩放反馈 | 100ms |
| 展开菜单 | 向上滑动 | 200ms |
| 导出菜单 | 淡入 | 150ms |
| 主题切换 | 渐变 | 300ms |

### 6.3 反馈机制

| 场景 | 反馈方式 |
|------|----------|
| 保存成功 | 底部 SnackBar "已保存" |
| 导出成功 | 系统分享面板 |
| 导出失败 | 弹窗提示错误 |
| 复制成功 | SnackBar "已复制" |
| 公式错误 | 红色错误提示 + 原始文本 |

## 8. 状态设计

### 8.1 空状态

```
┌─────────────────────────────────────────┐
│                                         │
│           📝                            │
│                                         │
│        还没有内容                        │
```

### 8.2 加载状态

| 场景 | 加载显示 |
|------|----------|
| 文档解析中 | 骨架屏 |
| 公式渲染中 | 公式位置显示浅蓝占位 |
| 导出中 | 全屏半透明遮罩 + 加载指示器 |

### 8.3 错误状态

| 场景 | 错误显示 |
|------|----------|
| 公式渲染失败 | 显示原始 LaTeX + 红色边框 + 提示 |
| 文件保存失败 | 弹窗 + 重试按钮 |
| 导出失败 | 错误详情 + 技术支持联系方式 |
